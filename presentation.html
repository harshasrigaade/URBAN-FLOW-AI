Key Feature 1

Multi-Objective Journey Planner

Rather than offering only the shortest path, our customized Dijkstra routing module solves for multiple goals simultaneously, matching individual commuter preferences.

4 Routing Priorities

Toggle dynamically between **Fastest** (low delay), **Cheapest** (minimal fare), **Safest** (high illumination), or **Eco-friendly** (minimized emissions).

Multi-Modal Integration

Seamless transitions across Bus, Metro, Train, Auto, Bike, and Walking tracks with integrated timings.

Balanced Balanced Mode

A hybrid algorithm that scores routes across all parameters to deliver the optimal overall journey.

Key Feature 4

Smart Cost Optimizer

Tackling transportation expense inflation by delivering transparent transit-fare comparisons and showing cost-to-time trade-offs.

✓
Transit Fare Aggregator

Aggregates municipal public rates (metro, buses, local commuter lines) with ride-sharing API estimates.

Cost vs. Time Trade-Off (Pareto Frontier)
